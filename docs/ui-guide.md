# UI Guide

## General Style

- Минималистичный аптечный стиль
- Чёткая иерархия информации
- Спокойные цвета, высокий контраст текста

## Layout

- Header: логотип, каталог, поиск, избранное, корзина
- Main: контент страницы
- Footer: контакты, информационные ссылки

## Catalog UX

- Desktop: категории слева, подкатегории справа
- Mobile: отдельный экран с категориями
- Переходы без перезагрузки

## Cards

- Карточка товара: изображение, название, бренд, цена
- Скидка отображается визуально
- Рецептурность помечается бейджем

## Pagination

- Кнопочная пагинация
- Фиксированное количество карточек на страницу

## Empty States

- Корзина пуста → текст + кнопка «На главную»
- Избранное пусто → текст + кнопка «На главную»

## Accessibility

- Семантическая разметка

## Performance

- Lazy loading редко используемых страниц
- Preload главных маршрутов
- Минимизация re-render
